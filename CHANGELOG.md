# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2025-01-30

### Added

- Initial release
- Multi-provider support (Anthropic, OpenAI, Google, Groq, xAI, Fireworks, Together, Moonshot, Venice)
- Web dashboard for managing agents
- Onboarding wizard for first-time setup
- Encrypted API key storage
- Settings page for managing provider keys
- Agent lifecycle management (create, start, stop, delete)
- Real-time chat with streaming responses
- Auto-download agent binary on first run
- SQLite database for persistence
